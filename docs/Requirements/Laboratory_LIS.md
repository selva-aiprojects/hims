
# Laboratory / Diagnostics System (LIS)

## Objective
Manage lab orders, results, and reporting.

## Scope
### Test Management
- Test catalog
- Panels

### Order Management
- Doctor orders
- Sample tracking

### Processing
- Technician workflow
- Result entry

### Reports
- PDF reports
- Digital signatures

## Integration
- Input: Visit
- Output: Lab results

## Detachability
- Can run as standalone Lab system
