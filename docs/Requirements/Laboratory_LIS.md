# Laboratory / Diagnostics System (LIS)

## Objective
Manage lab orders, results, and reporting.

## Subscription Tier
- **Standard** (Tier 2) or higher.

## Scope
### Test Management
- Catalog of tests
- Pricing

### Sample Tracking
- Collection status
- Barcoding

### Reporting
- Result entry
- Validation
- Digital report generation

## Integration
- Input: Doctor Order
- Output: Bill entry / Patient record
