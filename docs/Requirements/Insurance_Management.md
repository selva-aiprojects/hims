
# Insurance Management System

## Objective
Handle insurance workflows and claims.

## Scope
### Policy Management
- Provider details
- Patient policy

### Pre-Authorization
- Request submission
- Approval tracking

### Claims
- Claim creation
- Document upload

### Settlement
- Payment reconciliation

## Integration
- Input: Visit/Admission
- Output: Billing split

## Detachability
- Can run as standalone Claims system
